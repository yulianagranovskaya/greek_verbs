Greek Verb Conjugation Telegram Mini App

Files:
- index.html
- app.js
- verbs.json

How to run locally:
1. Put all 3 files in one folder.
2. Start a local server in that folder:
   python3 -m http.server 8000
3. Open:
   http://localhost:8000

How to publish:
- Upload the folder to Vercel / Netlify / any HTTPS hosting
- Set the Mini App URL in BotFather
